# 🚔 Chicago Crime Analysis & Power BI Dashboard

## 📌 Overview

This project focuses on analyzing crime data from Chicago to identify patterns, trends, and key insights. Using Excel for data preprocessing and Power BI for visualization, the project transforms raw data into an interactive dashboard for effective analysis.

## 🎯 Objectives

* Analyze crime trends over time
* Identify high-crime areas and categories
* Understand patterns based on location and time
* Build an interactive dashboard for data-driven insights

## 📊 Key Features

* Crime trend analysis over years
* Category-wise crime distribution
* Location-based insights
* Interactive Power BI dashboard

## 🛠️ Tools Used

* Microsoft Excel
* Power BI

## 🖥️ Dashboard

The Power BI dashboard provides interactive filters and visuals, allowing users to explore crime patterns dynamically and gain meaningful insights.

## 📁 Files

* Excel dataset and preprocessing
* Power BI dashboard file
* Project report

## 🚀 Outcome

The project demonstrates how data visualization tools can be used to analyze crime data and support informed decision-making.
